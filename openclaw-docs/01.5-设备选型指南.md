# 第 1.5 篇：设备选型指南

> **🎯 本篇目标**：帮你决定在什么设备上运行 OpenClaw

OpenClaw 可以在很多设备上运行，但不同设备的体验差异很大。选对了设备，后面会省很多心。

## 2. 操作系统支持情况

| 操作系统 | 支持情况 | 推荐程度 | 说明 |
|---------|---------|---------|------|
| **macOS** | ✅ 原生支持 | ⭐⭐⭐⭐⭐ | 最佳选择，安装最简单 |
| **Linux** | ✅ 原生支持 | ⭐⭐⭐⭐⭐ | 服务器首选，稳定可靠 |
| **Windows** | ⚠️ 需要 WSL2 | ⭐⭐⭐ | 不推荐直接安装，需通过 WSL2 运行 |

> ⚠️ **Windows 用户注意**：OpenClaw 不推荐直接在 Windows 环境安装。如果你用的是 Windows，需要先安装 WSL2（Windows Subsystem for Linux），在 Linux 子系统中运行。

---

## 3. 硬件配置要求

### 3.1 基础配置（使用云端 API）

如果你只用云端 API（如智谱、OpenAI、DeepSeek），对硬件要求很低：

| 组件 | 最低要求 | 推荐配置 |
|------|---------|---------|
| CPU | 2 核 | 4 核+ |
| 内存 | 2 GB | 4 GB+ |
| 存储 | 20 GB | 50 GB+ SSD |
| 网络 | 能上网 | 稳定网络 |

**结论：几乎所有能正常使用的电脑都满足要求。**

### 3.2 本地模型推理配置

如果你想在本地跑大模型（不用云端 API）：

| 模型大小 | GPU 显存要求 | 替代方案 |
|---------|-------------|---------|
| 7B 模型 | 8 GB+ 显存 | CPU 推理（很慢） |
| 13B 模型 | 16 GB+ 显存 | 不推荐 CPU 推理 |
| 更大模型 | 24 GB+ 显存 | 需要 RTX 3090/4090 或多卡 |

> 💡 **建议**：先用云端 API 把 OpenClaw 玩熟，再考虑本地模型。本地模型对硬件要求高，调试成本也高。

## 4. 设备方案对比

### 4.1 总览

| 方案 | 成本 | 维护难度 | 稳定性 | 适合人群 |
|------|------|---------|--------|---------|
| **Mac mini** | 中等 | 低 | ⭐⭐⭐⭐⭐ | 追求稳定、不想折腾 |
| **云服务器** | 低-中 | 中 | ⭐⭐⭐⭐ | 没有闲置设备、预算有限 |
| **闲置电脑** | 无 | 高 | ⭐⭐⭐ | 有旧电脑、爱折腾 |
| **NAS** | 高 | 高 | ⭐⭐⭐⭐ | 已有 NAS、想整合 |
| **本机临时** | 无 | 最低 | ⭐⭐ | 试用、临时使用 |

### 4.2 方案一：Mac mini（最推荐）

```
┌─────────────────────────────────────────────────────────────────┐
│                   Mac mini 方案                                  │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   ✅ 优点：                                                     │
│   • 功耗极低（10-20W），24小时运行不心疼电费                     │
│   • 体积小，放哪里都行                                          │
│   • macOS 原生支持，安装最简单                                  │
│   • 稳定性极高，可以连续运行几个月不重启                        │
│   • 可以配置为开机自启动                                        │
│                                                                 │
│   ⚠️ 注意：                                                    │
│   • 建议内存 16GB 起，跑本地模型建议 32GB                       │
│   • M1/M2/M3/M4 芯片都可以                                      │
│                                                                 │
│   💰 成本参考：                                                 │
│   • Mac mini M2 16GB：约 4000-5000 元                          │
│   • Mac mini M4 32GB：约 10000+ 元                             │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**适合：** 追求稳定、不想折腾、预算充足的用户。

### 4.3 方案二：云服务器

```
┌─────────────────────────────────────────────────────────────────┐
│                   云服务器方案                                   │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   ✅ 优点：                                                     │
│   • 不需要自己买硬件                                            │
│   • 公网 IP，不需要内网穿透                                     │
│   • 云厂商提供一键部署镜像                                      │
│   • 随时可以升级配置                                            │
│                                                                 │
│   ⚠️ 注意：                                                    │
│   • 需要一定的 Linux 运维基础                                   │
│   • 数据存储在云端，隐私敏感用户需考虑                          │
│   • 海外服务器访问 AI API 更稳定                                │
│                                                                 │
│   💰 成本参考（月费）：                                         │
│   • 腾讯云轻量 2核2GB：约 38 元/月                              │
│   • 腾讯云轻量 2核4GB：约 60 元/月                              │
│   • 阿里云轻量 2核2GB：约 50 元/月                              │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**云平台推荐：**

| 云平台 | 推荐程度 | 特点 |
|--------|---------|------|
| **腾讯云** | ⭐⭐⭐⭐⭐ | 性价比最高，带宽大，中文支持好 |
| **阿里云** | ⭐⭐⭐⭐ | 有官方 OpenClaw 一键部署镜像 |
| **华为云** | ⭐⭐⭐ | 企业级稳定性 |
| **海外 VPS** | ⭐⭐⭐⭐ | 访问 AI API 更稳定（如 Vultr、DigitalOcean） |

**适合：** 没有闲置设备、预算有限、有一定 Linux 基础的用户。

## 6. 各平台快速上手

### macOS

```bash
# 安装 Node.js
brew install node

# 安装 OpenClaw
npm install -g openclaw

# 验证
openclaw --version
```

### Linux（Ubuntu/Debian）

```bash
# 安装 Node.js
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt-get install -y nodejs

# 安装 OpenClaw
sudo npm install -g openclaw

# 验证
openclaw --version
```

### Windows（通过 WSL2）

```powershell
# 1. 启用 WSL2（管理员 PowerShell）
wsl --install

# 2. 重启电脑后，打开 Ubuntu 终端

# 3. 在 WSL2 中安装 Node.js 和 OpenClaw（同 Linux 步骤）
```

## 7. 我的推荐

| 你的情况 | 推荐方案 | 原因 |
|---------|---------|------|
| **苹果用户，追求稳定** | Mac mini M2/M4 16GB+ | 低功耗、稳定、省心 |
| **预算有限，想 24 小时在线** | 腾讯云轻量 2核4GB | 性价比高，稳定 |
| **有旧电脑，爱折腾** | 旧电脑 + Ubuntu Server | 零成本，学习 Linux |
| **只是试用一下** | 本机安装 | 最快上手 |
| **隐私敏感** | 本地设备（Mac mini / 旧电脑） | 数据不离开本地 |
