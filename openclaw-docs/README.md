## 适合谁

- ✅ 第一次接触 OpenClaw，想从 0 开始搭建
- ✅ 已经装上了，但遇到飞书、多 agent、共享群问题
- ✅ 想把 AI 接到自己工作流里的人
- ✅ 愿意动手排障、不排斥命令行

## 不适合谁

- ❌ 想要"装完马上能聊"的体验
- ❌ 不想碰配置文件和日志
- ❌ 不想维护本地目录和会话
